phpBB Extensions

Hier findest du eine Übersicht meiner phpBB Extensions. Solltest du Fehler finden oder Verbesserungsvorschläge haben kannst du mich gerne kontaktieren oder einfach einen Pull Request auf Github senden.

Demos meiner Extensions findest du in meinem Testforum.

phpBB Guestbook

Version: 0.1.2 Beta
Lizenz: GPL-2.0
Beschreibung: Diese Extension fügt ein einfaches Gästebuch zu deinem Forum hinzu.

Download Github


phpBB Textbox

Version: 0.1.0 Beta
Lizenz: GPL-2.0
Beschreibung: Diese Extension fügt eine Textbox in den Foren Header ein. Der Inhalt der Textbox kann im ACP geändert werden.

Download Github


phpBB Social Media

Version: 0.1.0 Beta
Lizenz: GPL-2.0
Beschreibung: Diese Extension fügt Buttons ein um Beiträge auf Social Media Platformen wie Facebook zu teilen.

Download Github
